    rntrc - v1.0.0

    rntrc-js

    Library for decoding and generating Ukrainian taxpayer identification numbers. Wikipedia

    Installation

    npm i rntrc

    Importing

    ES Modules

    import Rntrc from 'rntrc'

    CommonJS

    const Rntrc = require('rntrc')

    Browser (CDN)

    <script src="https://unpkg.com/rntrc@1.0.0/dist/index.umd.js"></script>

    Browser (module)

    <script type="module">
        import Rntrc from 'https://unpkg.com/rntrc@1.0.0/dist/index.js'
    </script>

    Usage

    Decoding

    const instance = new Rntrc(1234567899)

    console.log(
        instance.birthdate().toLocaleDateString(),
        instance.gender()
    )

    Generating

    const generated = Rntrc.generate({
        month: 4,
        gender: 'male'
    })

    console.log(generated)
